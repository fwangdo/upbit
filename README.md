# Upbit 클론 프로젝트

## 프로젝트 개요
React와 React Query를 사용하여 제작한 암호화폐 추적 및 가상 투자 웹 애플리케이션으로, Upbit 암호화폐 거래소 플랫폼의 주요 기능을 시뮬레이션합니다.

## 🚀 주요 기능
- [X] 실시간 상위 50개 암호화폐 가격 추적
- [X] 인터랙티브 가격 변동 대시보드
- [X] 상세 가격 이력 그래프
- [X] 가상 투자 시스템
- [X] 포괄적인 거래 이력
- [X] 자산 포트폴리오 추적

## 🛠 기술 스택
- React
- React Query
- JavaScript
- CodeGecko API
